## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router, Turbopack)
- TypeScript
- Tailwind CSS
- ESLint
- React Compiler

## Getting Started

Install dependencies and run the dev server:

```powershell
npm install
npm run dev
```

Visit `http://localhost:3000` to view the site locally.
